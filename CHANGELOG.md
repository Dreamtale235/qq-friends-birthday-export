# 更新日志

## v0.3.0 - 2026-05-25

- 优化异常路径下的浏览器关闭逻辑，减少后台进程残留。
- 优化已保存会话的检测流程，检测失败时也会释放 Playwright 资源。
- 修复 2 月 29 日生日在非闰年可能导致日期计算失败的问题。
- 导出 CSV 时自动避让同名文件，避免同一天多次导出覆盖旧结果。
- 新增 `unittest` 本地自检，覆盖星座、日期解析和生日倒计时边界。
- 完善 GitHub 上传前的忽略规则和调试说明。

## v0.2.0

- 支持 GUI 和 CLI 两种运行模式。
- 支持 QQ 邮箱扫码登录和会话保存。
- 支持遍历 1-12 月好友生日日历并导出 CSV。

## v0.1.0

- 初版：基于 Playwright 提取 QQ 邮箱日历中的好友生日信息。
